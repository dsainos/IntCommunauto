# Avance 1 — EDA (Communauto)

## Cómo correr
1. Crea una carpeta `data/` en el repo.
2. Coloca el archivo `Data Communauto 26-JAN-2026.xlsx` dentro de `data/`.
3. Abre y ejecuta el notebook:
   - `Avance1.Equipo_EDA_Communauto.ipynb`

## Notas
- El notebook está diseñado para ejecutarse **secuencialmente** (requisito del entregable).
- Renombra el notebook según indique tu profesor: `Avance1.#Equipo.ipynb`
